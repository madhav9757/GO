## Quick Start
1. Read **00-introduction.md** to understand Go and its benefits.
2. Follow **01-install-go.md** to install Go on Windows.
3. Set up your workspace with **02-go-workspace.md**.
4. Configure your editor using **03-editor-setup.md**.
5. Learn basic commands in **04-go-commands.md**.
6. Ensure environment variables are correct via **05-environment-variables.md**.
